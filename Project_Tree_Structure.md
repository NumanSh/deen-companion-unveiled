## 🏗️ Architecture Overview

This is a comprehensive Islamic companion application built with:
- **React 18** + **TypeScript**
- **Vite** for build tooling
- **Tailwind CSS** for styling
- **shadcn/ui** for UI components
- **React Router** for navigation
- **React Query** for data fetching
- **Vitest** for testing
- **Playwright** for E2E testing

The application features extensive Islamic content including Quran reading, prayer times, Hadith search, Athkar, and various AI-powered features for personalized Islamic learning and guidance.
